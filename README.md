# node-rclone-runner

## Install 

```
git clone https://github.com/shatfel/node-rclone-runner
```

## Build

```
npm run pre-build
npm run build
npm run post-build
```

## Config

Edit it in hash table *_C* _config.js_.

| Parameter | Description |
| --------- | ----------- |
| rcloneBin | path to rclone.exe |
| params | rclone`s parameters |
