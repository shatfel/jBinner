# jbinner

## Install 

```
git clone https://github.com/shatfel/jBinner
```

## Build

```
npm i pkg -g
npm i
```

## Config

Edit it in hash table __C_ in _jbinner.conf.js_.

| Parameter | Description |
| --------- | ----------- |
| rcloneBin | path to rclone.exe |
| params | rclone`s parameters |
